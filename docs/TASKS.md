# 微聊 AI 助手 — 任务清单

> 最后更新: 2026-04-06

## 如何使用

1. 每个 Agent（代理）接活前先读本文件，找自己要做的任务
2. 状态：待办 / 进行中 / 已完成
3. 做完一个任务，更新状态 + 简短备注（做了什么、遇到了什么）
4. 有新发现的 bug（程序缺陷）或改动，追加到本文件
